Bundled cursor themes
=====================

The "macOS Light", "macOS Dark", and "Linux (Sharp)" cursor themes are the
Bibata cursor set by Kaiz Khatri (ful1e5) and contributors.

  Project : https://github.com/ful1e5/Bibata_Cursor
  Version : v2.0.7 (Regular size), Windows .cur/.ani build
  License : GPL-3.0 and SIL Open Font License 1.1

  macOS Light   = Bibata Modern Ice      (white, rounded)
  macOS Dark    = Bibata Modern Classic  (black, rounded)
  Linux (Sharp) = Bibata Original Ice    (white, sharp corners)

Bibata is redistributed here unmodified under the terms of its license.
To add your own theme, drop a folder of .cur/.ani files (ideally with the
pack's install.inf) into this directory, or into:
    %APPDATA%\smooly\cursors\<Your Theme Name>\
and it will appear in smooly's Cursor theme list.


Themes built from Figma artwork
===============================

Themes generated by tools/build-cursors.ps1 from a Figma cursor pack must credit
the pack's author. Free Figma Community files are licensed CC BY 4.0, which
permits bundling and commercial use *with attribution*. Add a line here per pack:

  <Theme name> = built from "<Figma file title>" by <author>
                 Figma Community, CC BY 4.0
                 <figma file URL>

  e.g.
  Aqua = built from "Cursors for PC (mac + win)"
         Figma Community, CC BY 4.0
         https://www.figma.com/community/file/1218903777001900990/

Bundled Figma-built themes
--------------------------
HugeIcons = built from "Mouse Icons - Free Download (SVG Icons)" by Hugeicons
            Figma Community, CC BY 4.0
            https://www.figma.com/community/file/1485532265283423991/
            Rendered white-on-dark-halo for on-screen visibility (line-art
            outlines). Roles: pointer, link, text, help, unavailable, move,
            vertical/horizontal resize. Others fall back to the Windows default.

Note: if a pack's designs imitate Apple's macOS cursors, the Figma license
covers that file, but Apple's cursor design/trademark is separate — use a
neutral theme name rather than shipping it labelled "macOS".
